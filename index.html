Immutatsion

Arrayning immutatsion qilingan qilib, arrayning elementlarini o'zgartirishdan foydalanish uchun yana bir array qilishga imkon berilmaydi.

Arrayning elementlarini o'zgartirishda, arrayning yana bir array qilishga imkon berilgan qilib, mutatsionning qilingan arrayning yana bir array qilishga imkon berilmaydi. Bu yuqoridagi xabarlar qilingan arrayning yana bir array qilishga imkon berilmaydi.

concat() ikkita yoki undan ortiq massivlarni birlashtiradi.

concat() : ikkita yoki undan ortiq massivlarni birlashtiradi.

slice()() : massivdan qismni ajratib oladi va yangi massiv yaratadi..

map() : massivdagi har bir elementga amaliyot bajaradi va yangi massiv qaytaradi.

filter() : massivdagi ma'lum shartlarga mos keladigan elementlarni filtrlash.

creduce() : massivni bir nechta elementdan yagona qiymatga yig'ish.

reduceRight() : massivni o'ngdan chapga qarab yig'adi.

forEach() : massivdagi har bir elementni qayta ishlaydi, lekin yangi massiv yaratmaydi.

find() : massivdagi birinchi mos elementni topadi.

findIndex() : massivdagi birinchi mos elementning indeksini qaytaradi.

includes() : massivda ma'lum bir element bor-yo'qligini tekshiradi.

indexOf() : massivda ma'lum bir elementning indeksini topadi.

Mutatsion

Mutatsion metodlar massivning asl nusxasini (originalini) to'g'ridan-to'g'ri o'zgartiradi. Bu metodlar natijasida yangi massiv yaratilmaydi, balki mavjud massivga qo'shimcha o'zgarishlar kiritiladi.

Mutatsion metodlar massivga yangi elementlar qo'shish, mavjudlarini olib tashlash yoki tartibini o'zgartirish orqali o'zgaruvchan natija hosil qiladi. Bunda, massiv bilan bir vaqtda bajarilgan boshqa kodlar uchun massivning yangi holati ko'rinadi.

push() :massiv oxiriga element qo'shadi.

pop() : massiv oxiridan element olib tashlaydi.

shift() : massiv boshidan element olib tashlaydi.

unshift() : massiv boshiga element qo'shadi.

splice() : massivning o'rtasiga element qo'shadi yoki olib tashlaydi.

reverse() : massivni teskari tartibda o'zgartiradi.

sort() : massivni tartiblaydi.

Qidiruv metodlar

Qidiruv metodlari massivdagi ma'lum bir shartga yoki qiymatga mos keladigan elementni topish uchun ishlatiladi. Bu metodlar massiv ichida izlanayotgan elementni qaytarishi yoki uning indeksini berishi mumkin.

Qidiruv metodlari berilgan shartga ko'ra massivdagi barcha yoki ayrim elementlarni tekshiradi. Masalan, some() massivda hech bo'lmaganda bitta element shartga mos kelishini, every() esa barcha elementlarning shartga mosligini tekshiradi.

some() : massivdagi kamida bitta element shartga mos kelishini tekshiradi.

every() : barcha massiv elementlari shartga mos kelishini tekshiradi.

indexOf() : massivda elementning indexini qaytaradi.
